## Project Overview

This project analyzes **operational inefficiencies in telecom operators** using data analysis techniques. The goal is to identify patterns, bottlenecks, and behaviors that negatively impact performance, customer experience, or resource utilization.

The analysis is designed as a **portfolio-ready data analytics project**, with a strong emphasis on **business insight, clarity, and decision-making impact**.

---

## Objectives

* Identify operational inefficiencies in telecom operators
* Analyze relationships between key variables
* Transform raw data into **actionable business insights**
* Communicate findings through clear visuals and storytelling

---

## Key Questions Addressed

* Where are the main inefficiencies concentrated?
* Which variables show the strongest relationships?
* What patterns indicate operational or strategic issues?
* How could decision-makers act on these insights?
